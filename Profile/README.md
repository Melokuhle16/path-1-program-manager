# Program Manager System

## Description
This is a web-based Program Manager System that allows users to add, edit, delete, and search programs. The system helps manage program information efficiently.

## Features
- Add new programs
- Edit existing programs
- Delete programs
- Search programs
- Display program details
- User-friendly interface

## How to Run
1. Download or clone the project
2. Open the project in VS Code
3. Start the server
4. Open the application in a browser

## Technologies Used
- Java
- JSP
- HTML
- CSS
- JavaScript
